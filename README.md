# Heightmap To Table
A Streamlit-based tool for extracting elevation data from heightmap images. Users upload a grayscale heightmap, configure the grid resolution and min/max value range, and receive a structured table of XYZ coordinates mapped from pixel intensities. 

Useful for terrain analysis, GIS workflows, extraction data from existing visualizations and 3D modeling preparation.
